# Resistance
*Source: Player's Handbook (2024) p. 373. Available in the Free Rules (2024)* 

If you have Resistance to a [Damage Types](damage-types-xphb.md), damage of that type is halved against you (round down). Resistance is applied only once to an instance of damage.
