# Stable
*Source: Player's Handbook (2024) p. 374. Available in the Free Rules (2024)* 

A creature is Stable if it has 0 [Hit Points](hit-points-xphb.md) but isn't required to make [Death Saving Throw](death-saving-throw-xphb.md).
